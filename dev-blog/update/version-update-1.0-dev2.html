v1.0-dev2 업데이트 안내

2016년 1월 30일

CSHians 서비스가 1.0-dev2 버전으로 업데이트되었습니다.

이번 버전 업데이트는 1월 5주차 정기점검에서 이루어졌고, 사이트에 즉시 적용되었습니다.

이번 버전 업데이트에서는 다음과 같은 기능이 개선되었습니다.

  • XpressEngine 1.8.15 설치 및 초기 환경설정 수행
  • 호스팅 업체 변경
  • 도메인 연결
  • 게시판 정비
  • 블로그 리다이렉트 적용
  • 서비스 로고 업로드
  • 모바일 인터페이스 적용
이번 버전 업데이트에서는 다음과 같은 문제가 해결되었습니다.
  • 서브도메인 유무에 따른 로그인 풀림 문제 수정
개발 버전의 CSHians는 선공개용이며 사용하실 수 없습니다.

1.0-dev1 버전은 2015년 12월 22일에 창원과학고등학교 동아리 페스티발 현장에서 공개한 베타버전입니다. 착오 없으시기 바랍니다.

항상 커뮤니티 서비스 CSHians에 애정 어린 관심을 보내주신 데에 감사드립니다.
