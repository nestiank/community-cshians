v1.0 업데이트 안내

2016년 4월 24일

CSHians 서비스가 1.0 버전으로 업데이트되었습니다.

이번 버전 업데이트는 4월 3주차 정기점검에서 이루어졌으며, 사이트에 즉시 적용되었습니다.

이번 버전 업데이트에서는 다음과 같은 기능이 개선되었습니다.

  • 로그인 페이지 구문 수정
  • 접속 속도 개선 작업
  • 기타 서비스 개선 작업
이번 버전 업데이트에서는 다음과 같은 문제가 해결되었습니다.
  • 로그인 관련 문제
항상 커뮤니티 서비스 CSHians에 애정 어린 관심을 보내주신 데에 감사드립니다.
