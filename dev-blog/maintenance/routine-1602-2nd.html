2016년 2월 2주차 정기점검 안내

2016년 2월 5일

CSHians 2016년 2월 2주차 정기점검이 2016년 2월 9일부터 2월 14일까지 진행됩니다.

이 기간 동안 잠시 사이트 접속이 불가능하거나 사이트가 비정상적으로 동작할 수 있습니다.

이번 정기점검에서는 다음이 이루어집니다.

  • 메인 페이지 인터페이스 개선
  • 페이지 하단부 정비
  • 로그인 전용 페이지 적용
  • 웹메일서비스 개발 사전 작업
  • 통합검색에서 게스트로 로그인한 사용자가 자유게시판 이외의 게시물을 볼 수 있는 문제 수정
  • 통합검색에서 슬라이더가 표시되는 문제 수정
항상 커뮤니티 서비스 CSHians에 애정 어린 관심을 보내주신 데에 감사드립니다.
